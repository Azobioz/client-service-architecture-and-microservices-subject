# KT 1

## Что можно сделать

- Запустить консольное приложение, чтобы использовать todo функционал

## Какие инструменты я использовал при разработке 

- **Java**
- **Gradle**
- **Spring Boot Starter Web**
- **Spring Boot Starter Data JPA**
- **H2 Database**
- **Lombok**
- **Docker & Docker Compose**

## Что нужно для запуска
* Скачать Docker Desktop [здесь](https://www.docker.com/products/docker-desktop/)
* Скачать файл docker-compose.yml из репозитория 

### Локальный запуск с помощью Docker Compose

1. В командой строке перейдите на директорию с файлом docker-compose.yml
2. Пропишите в этой директории:
```
docker compose up -d todo-server 
```
3. Пропишите в той же директории:
```
docker compose run --rm todo-client  
```
чтобы запустить консольное приложение

В командной строке будет выводиться меню:
```
1. Показать задачи
2. Добавить задачу
3. Обновить статус задачи
4. Удалить задачу
5. Выход
```

## Завершить работу с приложением
1. В меню консольного приложения нажмите на 5
2. В командой строке перейдите на директорию с файлом docker-compose.yml
3. Прописать в командной строке: 
```
docker compose down
```
